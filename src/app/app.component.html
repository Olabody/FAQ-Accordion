FAQs

What is Frontend Mentor, how will it help me?

Frontend Mentor offers realistic coding challenges to help developers improve their frontend coding skills with projects in HTML, CSS and javaScript. It's suitable for all levels and ideal for portfolio building.

Is Frontend Mentor free?

Frontend Mentor is a free site, however there's a paid version which offers more priviledges than the free one.

Can I use Frontend Mentor projects on my portfolio?

Yes projects from Frontend Mentor can be added to your portfolio.

How can I get help if I'm stuck on a challenge?

There is a forum available for all users to ask questions about the project they are undertaking and asnswers are provided there to guide users.
